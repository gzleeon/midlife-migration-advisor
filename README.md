# 去留决策顾问

基于钱穆《中国历代政治得失》四层分析框架的 Claude Code 插件。通过对话式问卷评估政治风险与个人流动能力，联网查询最新移民政策，给出个性化建议与行动计划。

## 安装

```bash
git clone <仓库地址> ~/.claude/plugins/qian-mu-emigration-advisor
```

克隆后在 Claude Code 设置中启用插件，或重启 Claude Code 自动加载。

## 使用方式

### 方式一：斜杠命令

```
/emigration-assess
```

启动完整的对话式评估流程。

### 方式二：自然语言触发

在对话中提到"是否应该离开中国"、"移民建议"、"润去哪里"、"去留决策"等话题，Agent 会自动介入。

## 评估流程

**第一部分：外部风险感知**（钱穆四层）
- 制度层：政令可预期性、内部纠错机制
- 人事层：精英选拔逻辑、忠直之士的处境
- 经济层：地方财政压力、民间信心趋势
- 社会控制层：强制力成本、台海紧张感

**第二部分：个人情境**
- 年龄、职业、性格、家庭约束、语言能力、财务状况、主要动机

**联网查询**（自动执行）
- 目标国最新移民政策、配额、分数线
- 职业资质认证路径
- 近期政策变化

**输出报告**
- 钱穆四层风险诊断
- 去留建议（含历史类比）
- 目标国推荐 + 移民通道
- 职业落地策略
- 首要约束项与突破方案
- 三阶段行动计划

## 分析框架来源

钱穆《中国历代政治得失》核心方法论：
- 制度健康的标志是内建纠错机制（"门下省封驳"）
- 人事层衰败信号："进者皆佞幸，退者皆忠直"
- 经济层历史规律："并税简化后必然丛生加派"
- 控制层风险判断："兵权政权合一，天下无制衡"
