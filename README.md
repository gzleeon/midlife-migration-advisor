# 中年人移民顾问插件 · 基于国学大师钱穆的认知框架

基于国学大师钱穆对历代制度兴衰分析的 Claude Code 插件。通过对话式问卷评估外部环境变化与个人流动能力，联网查询最新移民政策，给出目标国推荐、职业落地策略与分阶段行动计划。

## 安装

```bash
git clone <仓库地址> ~/.claude/plugins/midlife-migration-advisor
```

克隆后在 Claude Code 设置中启用插件，或重启 Claude Code 自动加载。

## 使用方式

### 方式一：斜杠命令

```
/migration-assess
```

启动完整的对话式评估流程。

### 方式二：自然语言触发

在对话中提到"移民方向"、"海外定居"、"换个地方生活"、"子女教育移民"等话题，Agent 会自动介入。

## 评估流程

**第一部分：外部环境感知**（钱穆四层）
- 制度层：政策可预期性、规则稳定性
- 人事层：精英流动趋势、身边人的选择
- 经济层：民间信心、就业与财富环境
- 社会环境层：生活自由度、社会张力

**第二部分：个人情境**
- 年龄、职业、性格、家庭约束、语言能力、财务状况、主要动机

**联网查询**（自动执行）
- 目标国最新移民政策、配额、分数线
- 职业资质认证路径
- 近期政策变化

**输出报告**
- 钱穆四层环境诊断
- 移民建议（含历史类比）
- 目标国推荐 + 移民通道
- 职业落地策略
- 首要约束项与突破方案
- 三阶段行动计划

## 分析框架来源

国学大师钱穆《中国历代政治得失》核心认知方法：
- 制度健康的标志是内建纠错机制
- 人事环境的衰败先于整体崩溃显现
- 经济压力的转移方向预示下一阶段走势
- 强制性管控的成本趋势反映社会认同基础
